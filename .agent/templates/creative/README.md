# Creative Templates — Celebra-me

> **⚠️ These are Celebra-me-branded templates**, not multi-brand canonical templates. They reference
> `briefs/celebra-me.md` and are designed for Celebra-me creative production.

## Status

Estos templates son específicos de **Celebra-me** por ahora. Su contenido incluye referencias al
brand brief del producto y al tono/estilo de Celebra-me.

## Multi-brand Canonical Versions

Generic, multi-brand versions of these templates belong in the
[`creative-ops`](https://github.com/fm-dev-mx/creative-ops) workspace at `templates/creative/`. When
those are created, these Celebra-me-branded templates may be updated to reference the canonical
versions or be extracted generic.

## Templates

| Template                 | Purpose                                                |
| ------------------------ | ------------------------------------------------------ |
| `reel-brief.md`          | Short-form video reel script structure                 |
| `social-image-brief.md`  | Social post / carousel copy and image prompt structure |
| `video-frame-brief.md`   | Initial/final video frame prompt with ComfyUI params   |
| `image-prompt-output.md` | Generation parameter log for reproducibility           |
| `creative-qa-report.md`  | Checklist and report for reviewing creative outputs    |

## Cross-Workspace Reference

- Canonical generic templates (future): `creative-ops/templates/creative/`
- Shared creative infra (ComfyUI, pipelines): `creative-ops/workflows/comfyui/`
- Multi-brand brand briefs: `creative-ops/brands/`
