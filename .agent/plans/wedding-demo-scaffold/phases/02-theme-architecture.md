# Phase 02: Theme Development (Jewelry Box Wedding)

## Objectives

- Implement the "Jewelry Box" aesthetic tailored for weddings.
- Adhere to the **3-Layer Color Architecture**.

## Tasks

1. **Preset Creation**:
    - Create `src/styles/themes/presets/_jewelry-box-wedding.scss`.
    - Primary: Soft Ivory (`#FEFCF9`).
    - Accent: Champagne Gold (`#E5C387`).
    - Text: Charcoal Grey (`#333333`).

2. **Typography Setup**:
    - Primary Display (Names): `Pinyon Script`.
    - Secondary Display: `Playfair Display`.
    - Body: `EB Garamond`.

3. **Global Tokens**:
    - Map existing semantic tokens to the new wedding preset.
    - Define custom textures (e.g., subtle linen or silk paper).

4. **Variant Styling**:
    - Define shared `[data-variant="wedding"]` overrides for common components (Section, Container,
      Typography).

## Deliverables

- [ ] `src/styles/themes/presets/_jewelry-box-wedding.scss`.
- [ ] Integration of new preset into the main theme loading system.
